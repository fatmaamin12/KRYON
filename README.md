# KRYON Recovery Suite

A fresh, dependency-free prototype for the KRYON wellness brand site.

## Files

- `index.html` - site structure
- `styles.css` - visual system, responsive layout, and animations
- `script.js` - reveal effects, smooth navigation, product views, and finder state

## Run

Open `index.html` directly or serve the folder locally:

```bash
python3 -m http.server 4173
```
